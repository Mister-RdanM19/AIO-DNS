# ALLInOne-DNS

Tool **portable Windows** untuk mengatur **DNS-over-HTTPS (DoH)** langsung di **browser**  
(Chrome, Edge, Firefox) tanpa mengubah DNS sistem.

---

## Fitur
- Chrome, Edge, Firefox
- Menu pilihan DNS (1â€“4)
- Toggle OFF (kembali default)
- Auto-detect browser
- Log perubahan DNS
- Portable `.exe` (tanpa install)

---

## Pilihan DNS
1. Google DNS  
2. Cloudflare DNS  
3. Quad9 DNS  
4. AdGuard DNS  

---

## Cara Pakai
1. Jalankan `ALLInOne-DNS.exe` (**Run as Administrator**)
2. Pilih menu DNS
3. Restart browser

---

## Log
Perubahan DNS disimpan di:
`dns_log.txt`

---

## Catatan
- Tidak mengubah DNS sistem Windows
- Hanya mengatur DNS browser (DoH)
- Antivirus bisa memberi *false positive*
